          # 如何使用

          Valine-admin提供了自定义邮件模板的变量MAIL_TEMPLATE

          在leancloud添加这个变量即可

          • 添加一个Valine-admin的MAIL_TEMPLATE变量
          • 将上面给的源码，全部直接复制到右边即可
          • 重启应用来生效

          当有人收到回复的时候，就可以看到邮件模板了

          # 注意事项

          如果你无法成功设置此模板，那么请检查下面列出的清单

          • 自己使用了修改过的valine-admin，请使用原版Valine-admin
          • 自己原来能否正常接收或发生邮件
          • 有没有重启应用
